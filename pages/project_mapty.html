Mapty - activity log and track

about

Mapty is a project made as a practical example in Jonas Scmedtmann 's Javascript course, application that brings up live map using geolocation and enables user to select location and add activity data (running or cycling).

This template has a lot of potential, because it can be used for many other ideas like marking specific locations with pin and adding custom messages (shopping, bars and resurants, sightseeing, etc.)

features

  • Free resource maps
  • Geolocation
  • Centering map based on selected entry

planned development

  • Optimization for viewing on smaller devices
  • Making use of template for other purposes
  • Storing data localy for every user
