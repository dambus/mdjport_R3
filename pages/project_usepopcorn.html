about

React povered application for searching IMDB open database via OMDb API with feature to create list of watched movies and user rating.

Practice project in React course by Jonas Schmedtmann

features

Real-time search of open IMDB database (OMDb); Browsing movies & tv-shows with basic information; Browsing details of every movie on list; abillity to bookmark watched movies and rate them, making your own list with IMDB and personal rating; Calculating average rating as list expands and more

planned development

  • Including saving search for every user
  • UI redesign
  • Implementing design for mobile device
  • Adding new features regarding lists
  • Improving performance of live search
