about

React povered application for searching IMDB open database via OMDb API with feature to create list of watched movies, user rating and to-watch list

Inspired by practice project in React course by Jonas Schmedtmann.
Completly redesgined and with improved functionality.

features

Real-time search of open IMDB database (OMDb); Browsing movies & tv-shows with basic information; Browsing details of every movie on list; abillity to bookmark watched movies and rate them, making your own lists with IMDB and personal rating;

planned development

  • Adding new features regarding lists
  • Improving performance of live search
